## Strengths (keep doing this)

1. **Consistent numbering & mirroring.** Two-digit back/hole system with odd-left/even-right,
   and every play has a mirrored "Flip / Left" twin. This is excellent teaching architecture —
   players learn one play and get two.
2. **Clear offensive identity.** The self-labeled BASE plays (22 Trap, 22 Crunch, Tornado)
   show a defined "power-off-spread" identity. The scheme is not random.
3. **Smart constraint plays.** Fake 21 18 Bubble, Flip 28 6 RPO, and the unblocked-DE read
   note show real understanding of how to punish defenders for chasing the base run.
4. **Motion system is coherent.** Flip / Across / Jet are defined once and reused as tags
   rather than re-drawn per play.
5. **Blocking-scheme audible (Red/Purple).** Giving the line a color to reset protection is a
   genuinely advanced, sound idea.
6. **Formation-to-personnel logic.** Beast (heavy) and Doce (extra skill) give distinct
   run-heavy and spread-heavy personalities from the same base rules.

## Weaknesses (fix these)

1. **Too many plays to rep.** ~40+ named plays across 9 formations. Youth teams should
   carry **≤15 core plays**; reps win youth games, not menu depth.
2. **No written line assignments per play.** Slide 4 gives three generic scheme blurbs
   (Zone/Pass/Trap) but no play sheet says who blocks whom on *this* play. OLIQ can't grade
   what isn't written; players can't execute what they can't read.
3. **Cadence overload.** Four cadences (On One/Two/Set/Freeze) is a lot of pre-snap load
   for youth. Two + one change-up is plenty and cuts false starts.
4. **No special teams section.** No punt, PAT/kick, kickoff, or return. This is a full
   third of youth game snaps and it's missing.
5. **No situational organization.** No red-zone, short-yardage, two-minute, or
   "get-out-of-bounds" call sheet. Coaches need a "what do I call on 3rd-and-3" answer.
6. **Trick/low-percentage plays carried as if base.** Doce Cross Return Pass (double pass)
   and Banana Comeback are high-variance; fine as 1–2 gadget calls, risky as install volume.
7. **Diagrams carry the load, words don't.** The install lives in play-art; there's no
   text progression ("Week 1 install..."), so a new coach or sub can't run practice from it.

## IQ Module Notes

**QBIQ —** The QB is asked to do a lot: operate 4 cadences, execute zone reads (21/Read),
RPO give/keep/bubble decisions, jet-sweep mesh timing, and multiple play-action fakes with
backside protection responsibility ("Carter pickup backside blitz"). That's a high mental
load. Sound, but only if the read count per game plan is capped. Recommend: no more than
**2 live read/RPO concepts per week** until the QB is fluent.

**OLIQ —** The three taught schemes (Zone 45°/head-across, Pass shield/"don't create a hole,"
Trap pull) are technically sound and age-safe. **But there is no per-play blocking sheet.**
"22 Trap" and "23 Crunch 76" and "Tornado" reference protections that are never written down
as assignments. This is the #1 fix: one line of "who's got who" under every play.

**TeamIQ —** Scheme identity is consistent and strong (power/trap/tornado off spread motion).
Play selection supports the tendency well. Risk: with this many formations, film study will
show *you* nothing — you won't get enough reps in any one look to build a real tendency edge,
and you'll tip formation-to-play if the menu is too wide to disguise.

**MistakeIQ —** Highest baked-in risk plays: **Doce Cross Return Pass** (double pass —
turnover + illegal-forward-pass risk), **Banana Comeback / deep comebacks** (INT risk at
youth arm strength), **jet-sweep mesh** exchanges (fumble risk if timing is off), and
**pulling trap** vs quick youth DTs (TFL risk if the pull is slow). None are unsafe;
all are turnover-prone if under-repped — which argues again for a smaller, better-repped menu.
